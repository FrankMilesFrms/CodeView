## CodeView

# 安卓 代码编辑器 支持高亮、断点、大文本编辑等

    它支持放大镜、代码高亮、自定义代码、大文本编辑、断点调试、滑动条、主题切换等等。  
请配合**lexer**一起使用。

## 重要说明

    这里仅提供一个粗糙的例子，在后续开发中，我们将放弃对它的维护.....
      

>   1.但我们在它的基础之上，更新了更为流畅的编辑器控件,它着重于用户体验,更加强大的扩展插件能力和优化能力。  
>         2.实现了锁帧技术（在快速绘制的情况下，对内容进行优化以提高帧率）  
>         3.对光标操作和交互有了更深的判断，更加灵活。
